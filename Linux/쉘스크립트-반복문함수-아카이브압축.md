## 첫 번째 셸 스크립트

### 스크립트 파일 작성⭐

```bash
nano hello.sh
```

다음 내용을 입력한다.

```bash
#!/bin/bash
# 내 첫 번째 셸 스크립트입니다

echo "안녕하세요!"
echo "현재 시간: $(date)"
echo "저는 $USER 입니다"
```

저장: `Ctrl+O` → `Enter` → `Ctrl+X`

---

### 실행 권한 부여 + 실행

```bash
# 현재 권한 확인 (실행 권한 없음)
ls -l hello.sh
```

```bash
# 실행 권한 부여 (3주차 chmod 복습!)
chmod +x hello.sh
```

```bash
# 스크립트 실행
./hello.sh
```

```
출력 예시:
안녕하세요!
현재 시간: Mon Apr 20 09:10:23 KST 2026
저는 mirim 입니다
```

> 💡 실행할 때 반드시 **`./`** 을 앞에 붙여야 한다 <br>
> 그냥 `hello.sh` 만 치면 `command not found` 에러 <br>

## ⭐ 조건문 if

### if 문 기본 구조

```bash
if [ 조건 ]; then
    # 조건이 참일 때 실행
elif [ 다른조건 ]; then
    # 위는 거짓, 이 조건이 참일 때
else
    # 모든 조건이 거짓일 때
fi
```

> 💡 `[` 와 `]` 앞뒤에 **반드시 공백**이 있어야 한다  <br>
> 💡 if 의 끝은 **fi** (if를 거꾸로 쓴 것)

### 숫자 비교 연산자

| 연산자 | 의미 | 예시 |
| --- | --- | --- |
| `-eq` | 같다 (equal) | `[ $a -eq $b ]` |
| `-ne` | 다르다 (not equal) | `[ $a -ne $b ]` |
| `-lt` | 작다 (less than) | `[ $a -lt $b ]` |
| `-le` | 작거나 같다 | `[ $a -le $b ]` |
| `-gt` | 크다 (greater than) | `[ $a -gt $b ]` |
| `-ge` | 크거나 같다 | `[ $a -ge $b ]` |

## ⭐ 반복문 - for

### for 문 기본 구조

```bash
for 변수 in 목록; do
    # 반복할 명령어 ($변수 사용)
done
```

---

### 단순 목록으로 반복

```bash
nano for1.sh
```

```bash
#!/bin/bash
for fruit in 사과 바나나 포도 딸기; do
    echo "오늘의 과일: $fruit"
done
```

```
출력 예시:
오늘의 과일: 사과
오늘의 과일: 바나나
오늘의 과일: 포도
오늘의 과일: 딸기
```

---

### 숫자 범위로 반복

```bash
nano for2.sh
```

```bash
#!/bin/bash

# 방법 1: {시작..끝} 문법
for i in {1..5}; do
    echo "반복 $i번째"
done

echo "---"

# 방법 2: seq 명령어
for i in $(seq 1 3); do
    echo "seq: $i"
done
```

```
출력 예시:
반복 1번째
반복 2번째
반복 3번째
반복 4번째
반복 5번째
---
seq: 1
seq: 2
seq: 3
```

### 파일 목록으로 반복

## 아카이브와 압축

### tar란?

여러 파일을 **하나의 파일로 묶는** 명령어 <br>
기본적으로 묶기만 하고 압축은 안 함 → 압축 옵션을 같이 사용


### ⭐ tar 주요 옵션

| 옵션 | 의미 | 기억법 |
| --- | --- | --- |
| `c` | Create — 새 아카이브 만들기 | Create = 만들기 |
| `x` | eXtract — 아카이브 풀기 | eXtract = 추출 |
| `t` | lisT — 내용물 목록만 보기 | lisT = 목록 |
| `v` | Verbose — 진행 상황 표시 | Verbose = 상세히 |
| `f` | File — 파일 이름 지정 (필수!) | File = 파일 |
| `z` | gzip 압축/해제 (.gz) | - |
| `j` | bzip2 압축/해제 (.bz2) | - |

>💡 c, x, t 는 셋 중 하나만 쓸 수 있다 <br>
> f 옵션은 거의 항상 쓴다 → tar cvf / tar xvf / tar tvf 패턴으로 외우기

## 요약
| 명령어 | 기능 | 예시 |
| --- | --- | --- |
| `tar cvf` | 묶기 (Create) | `tar cvf a.tar 폴더/` |
| `tar czvf` | 묶기 + gzip 압축 | `tar czvf a.tar.gz 폴더/` |
| `tar xvf` | 풀기 (eXtract) | `tar xvf a.tar` |
| `tar xzvf` | 압축 해제 | `tar xzvf a.tar.gz` |
| `tar tvf` | 내용 확인 (lisT) | `tar tvf a.tar` |
| `gzip 파일` | 단일 파일 압축 | `gzip big.txt → big.txt.gz` |
| `gunzip 파일.gz` | 단일 파일 해제 | `gunzip big.txt.gz` |
| `zip -r` | zip 압축 (재귀) | `zip -r a.zip 폴더/` |
| `unzip 파일.zip` | zip 풀기 | `unzip a.zip` |
